# Mycelium Benchmarks

This folder contains reproducible benchmark suites for Mycelium.

## Benchmark Tracks

1. Discovery Benchmark
   - Keyword vs semantic search
   - Accuracy and latency

2. Messaging Benchmark
   - Concurrent requests
   - Throughput and failure rate

3. Workflow Benchmark
   - Multi-agent chain completion
   - End-to-end latency

4. Reliability Benchmark
   - Offline nodes
   - Timeout handling
   - Recovery

## Day 1 Target

- Generate 1,000 synthetic agent cards
- Generate 1,000 natural language queries
- Benchmark discovery accuracy and latency
- Save reproducible results to JSON
